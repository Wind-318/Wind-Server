# 访问地址：https://windserver.top
# 服务端架设流程
- 需要自行安装并启动 MySQL 和 Redis 服务端，需要配置 golang 环境
- 配置信息在 gofiles/config/config.go 中，需要根据情况自行填写；
- 需要自行添加 https 证书（或关闭 https）；
- 运行：go run main.go
